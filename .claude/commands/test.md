# Test Runner

Run the test suite for What The Game.

## Instructions

Run the appropriate tests based on the argument provided:

- No argument or `all`: Run full test suite with `bin/rails test`
- `system`: Run system tests with `bin/rails test:system`
- `models`: Run model tests with `bin/rails test test/models`
- `controllers`: Run controller tests with `bin/rails test test/controllers`
- A specific file path: Run that specific test file

After running tests:
1. Report the number of tests, assertions, failures, and errors
2. If there are failures, show the failure details
3. Suggest fixes for any failing tests

## Argument
$ARGUMENTS
